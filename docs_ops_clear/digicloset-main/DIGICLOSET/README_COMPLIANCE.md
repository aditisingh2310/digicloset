# Compliance, Encryption & Data Governance Setup

## Steps
1. Apply `encryption-policy.yaml` in your cluster.
2. Secure secrets using `secrets-config.yaml`.
3. Run `consent_tracking.sql` migration in Supabase SQL editor.
4. Review and customize `data-governance.md`.
5. Use `SOC2_Audit_Checklist.pdf` for compliance tracking.
