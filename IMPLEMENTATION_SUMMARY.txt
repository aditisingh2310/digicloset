GETTING STARTED
=================================================================

Step 1: Install dependencies
   pip install redis aioredis

Step 2: Start Redis
   docker run -d -p 6379:6379 redis:7-alpine

Step 3: Read quick start guide
   → QUICK_START_SECURITY.md

Step 4: Add middleware to your app
   See examples_integration.py for copy-paste code

Step 5: Update endpoints (gradual)
   Add Depends(get_shop_id) to /api/* routes
   Add PayloadValidator to AI/heavy endpoints
   Replace logger calls with safe_log()

Step 6: Test
   curl -H "X-Shop-ID: test" http://localhost:8000/api/products
   Verify rate limiting works
   Check logs for PII safety
   Confirm errors use new format

Step 7: Deploy
   See SECURITY_SETUP.md for production configuration
